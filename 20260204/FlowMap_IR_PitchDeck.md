## Slide 2: Problem

### 사고 도구의 구조적 한계

지식 작업자들은 매일 **사고의 딜레마**에 직면합니다.

| 기존 도구 유형 | 문제점 |
|---|---|
| **생산성 도구** (Notion, Docs) | 정형화된 구조를 강요 → 사고를 이미 정리된 상태로 전제 |
| **마인드맵 도구** (Miro, XMind) | 자유로운 발산은 허용하지만 사고를 지탱할 흐름 구조 부족 |
| **AI 챗봇** (ChatGPT, Claude) | 질문-응답 단발성 → 사고의 연속적 흐름을 유지하기 어려움 |

**핵심 문제: "정형성과 자율성 사이의 균형 부족"**

- 사고가 무한히 분산되거나, 고정된 계획에 갇히는 양극단
- 사고를 통제하지 않으면서도 안정적으로 머물게 하는 도구 부재
- 학습, 기획, 창작 과정에서 반복되는 인지적 과부하

## Slide 3: Solution

### Flow Map의 접근법

**"사고를 통제하지 않으면서도 안정적으로 머물게 하는 단계 기반 사고 인터페이스"**

Flow Map은 사용자가 입력한 주제에 대해 AI가 **4단계 사고 흐름 구조**를 제시합니다.

- **정답이 아닌 프레임** — 실행 계획이 아니라 사고가 머물 수 있는 인지적 프레임 제공
- **질문 중심 설계** — 각 단계가 질문으로 구성되어 사고를 자연스럽게 확장
- **시각적 집중 모드** — 선택한 단계만 선명하게, 나머지는 흐릿하게 → 인지 부하 최소화

**기존 도구와의 차이:**

```
기존: 빈 캔버스 → 사용자가 구조를 만듦 (높은 진입 장벽)
Flow Map: 주제 입력 → AI가 사고 구조를 제안 → 사용자가 탐색 (제로 진입 장벽)
```

## Slide 4: How It Works

### 사용자 여정 4단계

```
[1. 주제 입력] → [2. AI 구조 생성] → [3. 단계 탐색] → [4. 사고 확장]
```

**Step 1 — 주제 입력**
사용자가 메인 화면에서 프로젝트/학습 주제를 입력

**Step 2 — AI 구조 생성**
시스템이 입력 텍스트 기반으로 4단계 사고 흐름 구조를 자동 생성
- 각 단계: title + rationale + 3개 이상의 사고 확장 질문(prompts)

**Step 3 — 단계 탐색 (집중 모드)**
원형 노드 형태로 시각화된 단계를 클릭하여 집중 모드 진입
- 선택 단계는 강조, 나머지는 블러 처리
- 페이지 이동 없이 상태 변경 기반 UI 전환

**Step 4 — 사고 확장**
각 단계의 마인드맵과 질문을 탐색하며 사고를 깊이 있게 확장

## Slide 5: Product

### 핵심 기능 및 UI

**Core Features**

| 기능 | 설명 |
|---|---|
| **AI 사고 구조 생성** | 주제 입력만으로 4단계 전략적 사고 로드맵 자동 생성 |
| **원형 노드 시각화** | 단계별 원형 노드로 직관적 사고 흐름 표현 |
| **집중 모드 (Focus Mode)** | 선택 단계 강조 + 나머지 블러 → 인지 부하 최소화 |
| **AI 마인드맵 확장** | 각 단계에 대해 5개 브랜치 × 3개 하위 항목 마인드맵 자동 생성 |
| **데이터 영속화** | Supabase 기반 프로젝트/단계/프롬프트 저장 |

**UI/UX 원칙**
- 미니멀 디자인: 인지적 과부하를 유발하지 않는 클린 인터페이스
- 질문 중심: 실행 관리 도구가 아닌 사고 확장 도구로 인식되도록 설계
- 부드러운 전환: CSS 애니메이션 기반 자연스러운 상태 전환

## Slide 6: Market Size

### 시장 규모

**TAM (Total Addressable Market)**
- 글로벌 생산성 소프트웨어 시장: **$96.3B** (2026E)
- 개인 지식 관리 및 사고 도구 포함

**SAM (Serviceable Addressable Market)**
- 마인드맵 + 사고 구조화 도구 시장: **$4.8B**
- Miro, XMind, Notion, Obsidian 등이 경쟁하는 영역

**SOM (Serviceable Obtainable Market)**
- AI 기반 사고 구조화 도구 (신규 카테고리): **$480M**
- 초기 타겟: 한국어 사용 지식 작업자 → 아시아 → 글로벌 확장

**시장 트렌드**
- AI Native 도구에 대한 수요 급증 (2024~2026 CAGR 34%)
- "제2의 뇌" 시장의 성장: PKM(Personal Knowledge Management) 시장 확대
- 비구조적 사고 도구에서 구조화된 AI 보조 도구로의 전환

## Slide 7: Target Users

### 핵심 타겟 페르소나

**Primary — 기획자 & 창작자**
- 새로운 프로젝트를 시작할 때 사고의 출발점이 필요한 사람
- "무엇부터 생각해야 할지 모르겠다"는 문제를 자주 겪음

**Secondary — 학습자 & 연구자**
- 복잡한 주제를 단계적으로 이해하고 싶은 사람
- 교과서적 구조는 딱딱하고, 자유 노트는 산만한 사이에서 고민
- 대학생, 대학원생, 자기주도 학습자

**Tertiary — 전략 기획 실무자**
- 비즈니스 전략, 프로젝트 킥오프 시 사고 프레임이 필요한 PM/기획자
- 빠른 브레인스토밍과 구조화를 동시에 원하는 실무자
