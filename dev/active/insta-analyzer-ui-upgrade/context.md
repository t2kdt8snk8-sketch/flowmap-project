# Context

## 배경
- 사용자는 현재 인스타애널라이저 UI가 전반적으로 지나치게 둥글어서(Rounded) 전문성이 떨어진다고 피드백함.
- 전문 마케팅 분석 툴인 만큼, 신뢰감을 주는 직선 위주의 깔끔한 대시보드(Dashboard) 디자인으로 리뉴얼이 필요.

## 주요 버그 및 이슈
- **시장 트렌드 노란색 UI 잘림:** `시장 트렌드 탭` -> `2단계(심층 매트릭스)`의 테이블에 있는 노란색 바탕(`bg-amber-50`)의 '메인 팔레트' UI 텍스트가 부자연스럽게 잘리는 버그가 있음.
- **메인 탭 디자인 어색함:** 계정 정밀 분석 탭과 시장 트렌드 리서치 탭을 전환하는 메인 내비게이션 요소가 둥근 형태로 렌더링되어 다소 이상하고 비전문적으로 보임.
